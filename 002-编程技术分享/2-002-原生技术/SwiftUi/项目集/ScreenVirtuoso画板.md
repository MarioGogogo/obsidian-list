# ScreenVirtuoso

请帮设计进生城一个 macos 屏幕画板应用（Screen Drawing Board），
具体要求如下：
一、核心目标
开发一个可以悬浮在任意应用窗口之上的透明画板工具，用户可以在屏幕上直接绘制图形，而不影响底层原有页面的交互和显示内容。二、基础功能需求
支持如下绘制能力：
绘制矩形／正方形（可拖拽调整大小）绘制圆形/椭圆
绘制直线/箭头自由画笔（手绘模式）
文字输入（可编辑字体大小、颜色）三、交互方式
使用悬浮工具栏（FloatingTooLbar）类似截图标注类软件的交互方式鼠标拖拽绘制
支持撤销/重做支持清空画板四、窗口层级行为
画板采用透明窗口覆盖整个屏幕不遮挡底层 App 的显示
默认不拦截底层鼠标事件，仅在“绘制状态”时接管鼠标事件支持一键切换：
「穿透模式」（点击可穿透到下层应用）「编辑模式」（可绘制和选中图形）五、系统级能力（macos）

使用 macos 原生框架优先 
（Swift +  SwiftuI(优先) 或Appkit）
利用 NSwindow 透明窗口 +ignoresMouseEvents使用 CAShapeLayer / CoreGraphics
